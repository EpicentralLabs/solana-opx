## Getting Started

### Prerequisites
| Tool        | Version     |
|-------------|-------------|
| Node        | `>=22.14.0` |
| Rust        | `>=1.86.0`  |
| Anchor CLI  | `0.31.1`    |
| Solana CLI  | `>=2.1.21`  |
| mucho       | `0.10.0`    |

### Installation

#### Clone the repo

```shell
git clone <repo-url>
cd <repo-name>
```

#### Install Dependencies

```shell
pnpm install
```

#### Commands

Start the web app

```shell
pnpm dev
```

Build the web app

```shell
pnpm build
```
